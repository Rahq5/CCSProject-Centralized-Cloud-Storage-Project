# CCSProject-Centralized-Cloud-Storage-Project

# 1.0 Documentation (not completed yet...)
## 1.1 Introduction
- **Project Name :** CCSP or Centralized Cloud Storage Project
  
- **Description**: A Centralized cloud storage application that allows users to upload, download, and manage their files. initially hosting will be locally without using any server-less technologies like AWS , Azure and Google, instead will be hosted on a raspberry Pi device with NAS disks.
  
- **Technologies scope** :
	- Back-End language : Node.js 
	- Protocol : HTTP \ HTTPS
	- Storage : local Huawei D14 storage (my laptop)
	- Database : ??	
	- Authentications : ??
	- Other tools : 
		- **Rasberry PI**: as future main hosting device
		- **NAS disks** : storage for cloud service
		- **Deployment and Updating** : Docker

## 1.2 Project Goals
A list below shows what am personally trying to achieve while building this project:
- Store and retrieve files 
- File Sharing

## 1.3 Core Features
Here I will show what system **initially** should do :
- Upload Files
- Download Files
- Deletion Files
- Share Files
- view list of uploaded folders and files 
- view Size limit for each user 
- User registration and login 

## 1.3.1 Future Features
Here I will show what system in future should do :
- Centralized Server on raspberry pi device
- Larger storage capabilities (up to 3TB)
